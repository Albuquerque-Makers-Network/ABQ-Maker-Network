Required Pages

Creators' Network Required Pages

Landing Page

Dynamic landing page that will initially list all registered creators. The page will allow clients to filter creators based on products provided.

Type: dynamic

Creator Profile Page

Customers will be able to access this dynamic page by selecting a creator from the initial landing page. The profile page will provide more information about the creator and showcase a few portfolio items. Additionally a messaging feature will allow contact with the creator.

Type: dynamic

Creator Sign-up Page

This is a static form that will allow creators to sign-up for an account and list the products they provide.

Type: static

Creator Account Page

The creator account page will allow creators to modify the information that will appear on their profile page,as well as check their messages from potential clients.

Type: dynamic
