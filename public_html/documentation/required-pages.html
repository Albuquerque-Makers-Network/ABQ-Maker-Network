Makers' Network Required Pages

Landing Page

Dynamic landing page that will initially list all registered makers. The page will allow clients to filter makers based on their skills.

Type: dynamic

Sign-up Page

This is a static form that will allow makers and customers to sign-up for an account.

Type: static

Maker Account Page

The maker account page will allow makers to modify the information that will appear on their profile page and update their login info.

Type: dynamic

Community Account Page

The community account page will allow community users to update their log in information.

Type: dynamic

Maker Profile Page

Customers will be able to access this page by selecting a maker from the initial landing page. The profile page will provide more information about the maker and showcase a few portfolio items.

Type: dynamic
